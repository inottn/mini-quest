## 特性

- 使用 TypeScript 编写，提供完整的类型定义
- 支持 请求锁
- 支持 校验状态码
- 支持 请求／响应拦截器
- 支持 转换请求和响应数据
- 支持 上传／下载
- 支持 自定义适配器
- 适配 多端小程序

## 安装

使用 `pnpm` 安装:

```bash
pnpm add @inottn/miniapp-request
```

使用 `yarn` 或 `npm` 安装:

```bash
# 使用 yarn
yarn add @inottn/miniapp-request

# 使用 npm
npm i @inottn/miniapp-request
```
